## 🤖 AI对话真人化升级

### 三层防护体系

#### 1. System Prompt（15条硬规则）
```typescript
硬规则优化:
✅ 像群里随便聊，用第一人称，别主动报家门
✅ 被问是不是bot就嘴硬顶回去，别承认
✅ 只接当前这条消息，别翻旧账
✅ 接弹幕风格：短、快、准；别像客服排条目
✅ 别复读固定开头，有具体判断就别套口头禅
✅ 别用标题式开头（比如"结论""原因""建议""分析"）
⭐ 别用"作为AI/作为助手/根据我的理解/让我来帮你"这种AI腔
⭐ 别说"我理解你的意思/我明白你的问题/让我为你解答"
⭐ 直接说事，别铺垫，别客套
```

#### 2. Live Style Cue（21条风格提示）
```typescript
风格提示强化:
⭐ 别说"根据/基于/通过分析/综合来看"这种AI式开头
⭐ 别用"值得注意的是/需要指出的是/重要的是"
⭐ 别说"希望这个回答对你有帮助"结尾
✅ 直接说事，别铺垫开场白
✅ 别客服腔，不说"当然可以/希望帮到你"
✅ 像看到弹幕一样接，短点
✅ 少用emoji，主要靠语气和短句
```

#### 3. Reply Postprocess（13条后处理过滤）
```typescript
后处理过滤增强:
⭐ 过滤"根据|基于"开头
⭐ 过滤"让我(?:来)?|让我们"
⭐ 过滤"首先|其次|最后|第一|第二|第三"
⭐ 过滤"综合|总的来看说"
⭐ 过滤"这里|这边|我这边|我这里"
⭐ 过滤结尾"以上仅供参考|仅供参考"
⭐ 过滤结尾"祝你|祝您...愉快|顺利|成功"
✅ 过滤"作为AI/人工智能/语言模型"
✅ 过滤"我会|我来|我将...整理|分析|总结"
```

## 🖼️ 图片爬取脚本

### 1. CS选手图片爬取 (`scripts/fetch-player-images.js`)

**功能:**
- 从 Liquipedia 和 Wikimedia 爬取选手高清图片
- 支持 10 位顶级选手（ZywOo、s1mple、donk、m0NESY等）
- 自动去重、尺寸过滤、延迟控制

**使用:**
```bash
cd /path/to/qqbot
node scripts/fetch-player-images.js
```

**特性:**
- 目标：每位选手200+张图片
- 延迟：2秒/请求，避免被封IP
- 过滤：最小50KB，过滤缩略图
- 来源：Liquipedia Gallery + Wikimedia Commons API

### 2. 木柜子角色图片爬取 (`scripts/fetch-mokoko-images.js`)

**功能:**
- 从 Bestdori API 爬取 MyGO/Ave Mujica 角色卡面
- 支持 12 位角色（高松燈、長崎素世、椎名櫻子等）
- 训练前、训练后、特训后多版本

**使用:**
```bash
cd /path/to/qqbot
node scripts/fetch-mokoko-images.js
```

**特性:**
- 目标：每位角色200+张卡面
- 来源：Bestdori 官方 API
- 版本：normal、trained、rip三个版本
- 支持：MyGO!!!!! 和 Ave Mujica 两个乐队
